= Dockle Analyzer Report
== Properties
* [[properties_analyzer]]**`analyzer`** *(string, required)*: Must be: `"dockle"`.
* [[properties_repository]]**`repository`** *(string, required)*
* [[properties_tag]]**`tag`** *(string, required)*
* [[properties_error]]**`error`** *(string)*: Any error encountered during execution.
* [[properties_passed]]**`passed`** *(boolean, required)*: True if no issues were found, False otherwise.
* [[properties_issues_count]]**`issues_count`** *(integer, required)*: Total number of issues found.
* [[properties_issues_by_level]]**`issues_by_level`** *(object, required)*: Count of issues grouped by severity level. Cannot contain additional properties.
** [[properties_issues_by_level_properties_FATAL]]**`FATAL`** *(integer)*: Default: `0`.
** [[properties_issues_by_level_properties_WARN]]**`WARN`** *(integer)*: Default: `0`.
** [[properties_issues_by_level_properties_INFO]]**`INFO`** *(integer)*: Default: `0`.
** [[properties_issues_by_level_properties_SKIP]]**`SKIP`** *(integer)*: Default: `0`.
** [[properties_issues_by_level_properties_PASS]]**`PASS`** *(integer)*: Default: `0`.
* [[properties_issues]]**`issues`** *(array, required)*: List of issues found by Dockle.
** [[properties_issues_items]]**Items** *(object)*
*** [[properties_issues_items_properties_code]]**`code`** *(string, required)*: Dockle rule code (e.g., CIS-DI-0001).
*** [[properties_issues_items_properties_title]]**`title`** *(string, required)*: Short description of the rule.
*** [[properties_issues_items_properties_level]]**`level`** *(string, required)*: Severity level of the issue. Must be one of: "FATAL", "WARN", "INFO", "SKIP", or "PASS".
*** [[properties_issues_items_properties_alerts]]**`alerts`** *(array, required)*: Specific details or files related to the issue.
**** [[properties_issues_items_properties_alerts_items]]**Items** *(string)*
