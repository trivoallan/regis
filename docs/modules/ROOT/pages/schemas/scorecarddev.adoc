= Scorecard Analyzer Report
*Report containing OpenSSF Scorecard security assessment for the image source repository.*
== Properties
* [[properties_analyzer]]**`analyzer`** *(string, required)*: Must be: `"scorecarddev"`.
* [[properties_repository]]**`repository`** *(string, required)*: Docker image repository path.
* [[properties_source_repo]]**`source_repo`** *(string or null, required)*: Resolved source code repository URL, or null if not found.
* [[properties_scorecard_available]]**`scorecard_available`** *(boolean, required)*: Whether OpenSSF Scorecard data was successfully retrieved.
* [[properties_score]]**`score`** *(number or null, required)*: Overall Scorecard score (0-10), or null if unavailable.
* [[properties_checks]]**`checks`** *(array, required)*: Individual Scorecard check results.
** [[properties_checks_items]]**Items** *(object)*: Cannot contain additional properties.
*** [[properties_checks_items_properties_name]]**`name`** *(string, required)*: Check name (e.g. Maintained, Code-Review, Vulnerabilities).
*** [[properties_checks_items_properties_score]]**`score`** *(integer, required)*: Check score (-1 to 10). -1 means not applicable.
*** [[properties_checks_items_properties_reason]]**`reason`** *(string, required)*: Human-readable explanation of the score.
