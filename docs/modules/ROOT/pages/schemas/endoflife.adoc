= End-of-Life Analyzer Report
*Report containing lifecycle information from endoflife.date.*
== Properties
* [[properties_analyzer]]**`analyzer`** *(string, required)*: Must be: `"endoflife"`.
* [[properties_repository]]**`repository`** *(string, required)*
* [[properties_product]]**`product`** *(string, required)*: Product slug used to query endoflife.date.
* [[properties_product_found]]**`product_found`** *(boolean, required)*: Whether the product was found on endoflife.date.
* [[properties_tag]]**`tag`** *(string, required)*: Image tag that was analyzed.
* [[properties_matched_cycle]]**`matched_cycle`** *(object or null, required)*: Release cycle matching the image tag, or null if no match. Cannot contain additional properties.
** [[properties_matched_cycle_properties_cycle]]**`cycle`** *(string)*
** [[properties_matched_cycle_properties_release_date]]**`release_date`** *(string or null)*
** [[properties_matched_cycle_properties_eol]]**`eol`** *(string or boolean)*
** [[properties_matched_cycle_properties_latest]]**`latest`** *(string or null)*
** [[properties_matched_cycle_properties_latest_release_date]]**`latest_release_date`** *(string or null)*
** [[properties_matched_cycle_properties_lts]]**`lts`** *(boolean)*
* [[properties_is_eol]]**`is_eol`** *(boolean or null, required)*: Whether the matched cycle has reached end-of-life. Null if no match.
* [[properties_active_cycles_count]]**`active_cycles_count`** *(integer or null, required)*: Number of currently supported release cycles. Minimum: `0`.
* [[properties_eol_cycles_count]]**`eol_cycles_count`** *(integer or null, required)*: Number of end-of-life release cycles. Minimum: `0`.
