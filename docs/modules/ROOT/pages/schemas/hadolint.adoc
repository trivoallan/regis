= Hadolint Analyzer Report
== Properties
* [[properties_analyzer]]**`analyzer`** *(string, required)*: Must be: `"hadolint"`.
* [[properties_repository]]**`repository`** *(string, required)*
* [[properties_tag]]**`tag`** *(string, required)*
* [[properties_error]]**`error`** *(string)*: Any error encountered during execution.
* [[properties_passed]]**`passed`** *(boolean, required)*: True if no issues were found, False otherwise.
* [[properties_issues_count]]**`issues_count`** *(integer, required)*: Total number of issues found.
* [[properties_issues_by_level]]**`issues_by_level`** *(object, required)*: Count of issues grouped by severity level. Cannot contain additional properties.
** [[properties_issues_by_level_properties_error]]**`error`** *(integer)*: Default: `0`.
** [[properties_issues_by_level_properties_warning]]**`warning`** *(integer)*: Default: `0`.
** [[properties_issues_by_level_properties_info]]**`info`** *(integer)*: Default: `0`.
** [[properties_issues_by_level_properties_style]]**`style`** *(integer)*: Default: `0`.
* [[properties_issues]]**`issues`** *(array, required)*: List of issues found by Hadolint.
** [[properties_issues_items]]**Items** *(object)*
*** [[properties_issues_items_properties_code]]**`code`** *(string, required)*: Hadolint rule code (e.g., DL3008).
*** [[properties_issues_items_properties_level]]**`level`** *(string, required)*: Severity level of the issue. Must be one of: "error", "warning", "info", or "style".
*** [[properties_issues_items_properties_message]]**`message`** *(string, required)*: Description of the issue.
*** [[properties_issues_items_properties_line]]**`line`** *(integer or null)*: Line number in the pseudo-Dockerfile.
